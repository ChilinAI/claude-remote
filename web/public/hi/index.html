कैसे काम करता है

अपने Mac को रिमोट कंट्रोल करने के तीन आसान स्टेप्स

1

ऐप इंस्टॉल करें

अपने Mac पर Claude Remote डाउनलोड करें। ऐप मेन्यू बार में रहता है और आपके अकाउंट से कनेक्ट होता है।

2

चैट खोलें

अपने फोन या किसी भी ब्राउज़र से clauderemote.web.app पर जाएँ। लॉगिन करें और टाइप करना शुरू करें।

3

Claude एक्जीक्यूट करता है

आपका मैसेज तुरंत Mac पर पहुँचता है। Claude Code टास्क को एक्जीक्यूट करता है और रियल-टाइम में रिजल्ट स्ट्रीम करता है।

विशेषताएँ

रिमोट AI कोडिंग के लिए जो कुछ भी चाहिए

📱

मोबाइल एक्सेस

iPhone या Android से Claude Code के साथ फुल चैट। रेस्पॉन्सिव डिज़ाइन, किसी भी स्क्रीन पर आरामदायक।

⚡

तुरंत डिलीवरी

Firebase Realtime Database — मैसेज तुरंत डिलीवर होते हैं। रियल-टाइम में रेस्पॉन्स स्ट्रीमिंग।

🔒

एंड-टू-एंड एन्क्रिप्शन

सभी संदेश भेजने से पहले ब्राउज़र में एन्क्रिप्ट होते हैं और केवल आपके Mac पर डिक्रिप्ट होते हैं। ECDH P-256 कुंजी विनिमय + AES-256-GCM — सर्वर भी डेटा प्लेनटेक्स्ट में नहीं देख सकता।

💻

लाइटवेट ट्रे ऐप

न्यूनतम फुटप्रिंट — मेन्यू बार में बस एक आइकन। ऑटो-स्टार्ट, ऑटो-अपडेट, लाइव स्टेटस इंडिकेटर।

अभी शुरू करें

Mac पर ऐप डाउनलोड करें और कहीं से भी Claude Code को कंट्रोल करें।

 Mac के लिए डाउनलोड

इंस्टॉलेशन

App Store के बाहर से डाउनलोड किए गए ऐप्स के लिए macOS चेतावनी दिखा सकता है

1

DMG खोलें

डाउनलोड की गई .dmg फाइल पर डबल-क्लिक करें और Claude Remote को Applications फोल्डर में ड्रैग करें।

2

सेटिंग्स में अनुमति दें

ऐप खोलने का प्रयास करें — macOS इसे ब्लॉक करेगा। System Settings → Privacy & Security पर जाएँ, नीचे स्क्रॉल करें और Open Anyway क्लिक करें। अपना पासवर्ड दर्ज करें।

3

हो गया!

इसके बाद ऐप सामान्य रूप से खुलता है। macOS आपकी पसंद याद रखता है — कोई अतिरिक्त स्टेप्स नहीं।
