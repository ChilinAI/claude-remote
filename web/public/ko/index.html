어디서든 Claude Code에 접근

어디서든 Mac에 코딩 작업을 보내세요. Claude Code는 컴퓨터에서 실행되고, 스마트폰이나 브라우저에서 제어합니다.

 Mac용 다운로드 웹 채팅 열기 →

사용 방법

Mac을 원격 제어하는 3단계

1

앱 설치

Mac에 Claude Remote를 다운로드하세요. 앱은 메뉴 막대에 상주하며 계정에 연결됩니다.

2

채팅 열기

스마트폰이나 브라우저에서 clauderemote.web.app을 방문하세요. 로그인하고 작업을 입력하세요.

3

Claude가 실행

메시지가 즉시 Mac에 도달합니다. Claude Code가 작업을 실행하고 실시간으로 결과를 스트리밍합니다.

기능

원격 AI 코딩에 필요한 모든 것

📱

모바일 접근

iPhone이나 Android에서 Claude Code와 풀 채팅. 반응형 디자인으로 어떤 화면에서도 편리합니다.

⚡

즉시 전달

Firebase Realtime Database — 메시지가 즉시 전달됩니다. 실시간 응답 스트리밍.

🔒

종단간 암호화

모든 메시지는 브라우저에서 암호화된 후 전송되며 Mac에서만 복호화됩니다. ECDH P-256 키 교환 + AES-256-GCM — 서버조차 데이터를 평문으로 볼 수 없습니다.

💻

가벼운 트레이 앱

최소한의 설치 공간 — 메뉴 막대에 아이콘 하나만. 자동 시작, 자동 업데이트, 실시간 상태 표시.

지금 시작하세요

Mac에 앱을 다운로드하고 어디서든 Claude Code를 제어하세요.

 Mac용 다운로드

설치

App Store 외부에서 다운로드한 앱에 대해 macOS가 경고를 표시할 수 있습니다

1

DMG 열기

다운로드한 .dmg 파일을 더블클릭하고 Claude Remote를 응용 프로그램 폴더로 드래그하세요.

2

설정에서 허용

앱을 열려고 하면 macOS가 차단합니다. 시스템 설정 → 개인정보 보호 및 보안으로 이동하여 아래로 스크롤한 후 확인 없이 열기를 클릭하세요. 비밀번호를 입력하세요.

3

완료!

이후 앱은 정상적으로 열립니다. macOS가 선택을 기억합니다 — 추가 단계가 필요 없습니다.
