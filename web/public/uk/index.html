Керуй Claude Code з будь-якого пристрою

Відправляй завдання на свій Mac з будь-якого місця. Claude Code працює на твоєму комп'ютері — ти керуєш з телефону або браузера.

Як це працює

Три прості кроки для віддаленого керування Mac

1

Встанови застосунок

Завантаж Claude Remote на Mac. Застосунок живе в треї та підключається до твого акаунту.

2

Відкрий чат

Зайди на clauderemote.web.app з телефону або будь-якого браузера. Увійди та почни писати.

3

Claude виконує

Твоє повідомлення миттєво потрапляє на Mac. Claude Code виконує завдання та повертає результат у реальному часі.

Можливості

Все що потрібно для віддаленої роботи з Claude Code

📱

Мобільний доступ

Повноцінний чат з Claude Code прямо з iPhone або Android. Адаптивний дизайн, зручно на будь-якому екрані.

⚡

Миттєва доставка

Firebase Realtime Database — повідомлення доставляються миттєво. Стрімінг відповідей у реальному часі.

🔒

Наскрізне шифрування

Усі повідомлення шифруються у браузері перед відправкою та розшифровуються лише на твоєму Mac. Обмін ключами ECDH P-256 + AES-256-GCM — навіть сервер не бачить дані у відкритому вигляді.

💻

Легкий tray-застосунок

Мінімальна присутність — іконка в треї. Автозапуск, автооновлення, індикатор статусу.

Почни зараз

Завантаж застосунок на Mac та керуй Claude Code звідки завгодно.

 Завантажити для Mac

Встановлення

macOS може показати попередження для застосунків не з App Store

1

Відкрий DMG

Двічі клікни на завантажений .dmg файл та перетягни Claude Remote у папку Applications.

2

Дозволь у налаштуваннях

Спробуй відкрити — macOS заблокує. Зайди в Системні налаштування → Конфіденційність та безпека, прокрути вниз та натисни Все одно відкрити. Введи пароль.

3

Готово!

Після цього застосунок відкривається як зазвичай. macOS запам'ятовує твій вибір — більше нічого робити не потрібно.
