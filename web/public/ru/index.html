Как это работает

Три простых шага — и ты управляешь своим Mac удалённо

1

Установи приложение

Скачай Claude Remote на Mac. Приложение живёт в трее и подключается к твоему аккаунту.

2

Открой чат

Зайди на clauderemote.web.app с телефона или любого браузера. Залогинься и начни писать.

3

Claude выполняет

Твоё сообщение мгновенно попадает на Mac. Claude Code выполняет задачу и возвращает результат.

Установка

macOS может показать предупреждение для приложений не из App Store

1

Открой DMG

Дважды кликни на скачанный .dmg файл и перетащи Claude Remote в папку Applications.

2

Разреши в настройках

Попробуй открыть — macOS заблокирует. Зайди в Системные настройки → Конфиденциальность и безопасность, прокрути вниз и нажми Всё равно открыть. Введи пароль.

3

Готово!

После этого приложение открывается как обычно. macOS запоминает твой выбор — больше ничего делать не нужно.
